

# Clase 12: Componentes de Hardware y Almacenamiento 


En esta práctica aprendí a interactuar con los componentes físicos del sistema desde la terminal, enfocándome en el monkitoreo de recursos crfíticos


## Objetivos cumplidos:

* **Análisis de Alamcenamiento:** Uso de comando 'df' para gestionar particiones y espacio disponible.
* **Gestión de Memoria:** Monitoreo de memoria RAM física y Swap mediante el comando 'free'.
* **Identificación de CPU:** Extracción de metadatos del procesador con 'lscpu.
* **Lógica de Alertas:** Implementación de un sistema de avisos automáticos cuando el uso de disco supera el ubmbral del 80%.


## Script desarrollo: 'monitor_hardware.sh'
 Un script de shell que automatiza el reporte de salud del hardware, proporcionando una visión rápida del estado del servidor.
