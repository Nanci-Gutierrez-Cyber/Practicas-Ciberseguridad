## Clase 8: Gestión de Archivos y Empaquetamiento 
   En esta sesión practiqué la administración avanzada de archivos de Linux, enfocada en la organizacion de evidencias de ciberseguridad
**Habilidades demostradas:**
* **Estructura:** Creación de directorios jerárquicos para reportes y herramientas.
* **Permisos:** uso de 'chmod' para asegurar reportes sensibles ('600') y habilitar scripts de escaneo ('+x').
* **Empaquetamiento:** Uso de 'tar' con compresión 'gzip' para la exfiltración o respaldo de datos.
* **Automatización:** Manejo de flujos de trabajo con Git y GitHub
